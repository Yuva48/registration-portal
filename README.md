# Registration Portal

Professional multi-step registration form with modern design.

## Features
- Multi-step form wizard
- File upload support  
- Form validation
- Responsive design
- Success confirmation

## Files
- `index.html` - Main registration form
- `styles.css` - Styling and design
- `script.js` - Form validation and interactions
- `process-simple.php` - Backend form processing
- `success.php` - Success confirmation page
- `index.php` - Application entry point
